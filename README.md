# Merchant Holdings LLC — website

Single-page static site for Merchant Holdings LLC. Plain HTML + CSS, no build step, no dependencies.

- **`index.html`** — the entire site (self-contained: inline styles, inline SVG favicon, responsive, light/dark aware).
- Deploys as-is via **GitHub Pages** (Settings → Pages → deploy from `main` / root).

## Before going live
- Replace the placeholder contact email (`contact@merchantholdings.com`) with a real business inbox. Do **not** use the GridMail sending inbox here.
